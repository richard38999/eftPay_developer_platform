## 交易状态说明
| 状态码 | 描述 | 说明 |
|--------|------|------|
| SUCCESS | 交易成功 | 交易已成功完成 |
| PROCESSING | 处理中 | 交易正在处理中 |
| FAILED | 交易失败 | 交易失败 |

| VOIDED | 已撤销 | 交易已被撤销 |

| PARTIAL_REFUNDED | 部分退款 | 交易已部分退款 |

## 注意事项
1. 交易处理中（01）状态需要调用查询接口获取最终结果
2. 系统错误码（96-99）建议重试交易
3. 业务错误码（B1-B6）需要检查请求参数
4. 支付渠道错误码需要根据具体渠道处理
5. 交易状态为最终状态时不可更改
